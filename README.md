quick and dirty typescript + vite web boilerplates

1. clone
   ```shell
   # vanilla + tailwind
   npx degit kavsingh/tsblank
   # solidjs + tailwind
   npx degit kavsingh/tsblank#solid-js
   # react dom + tailwind
   npx degit kavsingh/tsblank#react-dom
   ```
2. run
   ```shell
   bun i && bun dev
   ```
